Simple Alert



There are five variants of Simple Alerts available, success alert, info alert, warning alert, and error alert. Simple Alerts can be created by adding the .alert class and as per your requirement for the variant, you will have to add .alert-success, .alert-info, .alert-warn, .alert-error class names.
